# HandGesture

Realtime gesture tangan menggunakan **Mediapipe + OpenCV**.

## Fitur

- Deteksi gesture: Halo, OK, I Love You, Peace, Fist
- Skeleton tangan garis merah + titik hitam
- Teks gesture muncul di layar
